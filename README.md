# 🛒SpringBoot Project-Shopping Mall Web
스프링부트 + JPA - '쇼핑몰 웹사이트'</br></br>

## 💻프로젝트 소개
의류 구매를 위한 쇼핑몰 웹사이트입니다.</br></br>

## 📆개발 기간
22.09.05 - 22.12.17</br></br>

## ⚙️개발 환경
IDE - InteliJ

Build Tool - Maven

SpringBoot - 2.7.1

Database - MySQL</br></br>

## 📌주요 기능
##### 회원
- 회원가입 및 로그인
##### 상품
- 상품 등록, 관리, 수정, 조회, 상세 페이지
##### 주문
- 상품 주문, 주문 내역 조회, 주문 취소
##### 장바구니
- 장바구니 담기, 조회, 삭제, 주문</br></br>
